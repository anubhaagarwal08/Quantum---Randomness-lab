# Quantum Randomness Lab

## Overview

This project demonstrates the difference between classical random number generation and quantum random number generation using Python and Qiskit.

The project explores how quantum measurement produces inherently random outcomes and compares them with pseudo-random numbers generated by classical algorithms.

---

## Features

- Classical random number generation
- Quantum random number generation (QRNG)
- Histogram visualization
- Statistical comparison
- Probability distribution analysis

---

## Technologies Used

- Python
- Qiskit
- Qiskit Aer
- Matplotlib
